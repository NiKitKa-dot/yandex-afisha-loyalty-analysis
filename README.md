# Анализ лояльности пользователей Яндекс Афиши

## Описание проекта
Аналитическое исследование поведения пользователей сервиса Яндекс Афиша с целью выявления факторов, влияющих на повторные покупки.

## Структура проекта
- `project.ipynb` - основной файл с анализом
- `requirements.txt` - зависимости
- `.env` - переменные окружения (не включать в репозиторий)
- `.gitignore` - список исключаемых файлов

## Установка и запуск
1. Клонировать репозиторий
2. Установить зависимости: `pip install -r requirements.txt`
3. Создать файл `.env` с параметрами подключения к БД
4. Запустить Jupyter: `jupyter notebook`
5. Открыть `project.ipynb`

## Ключевые результаты
- Выявлены сегменты пользователей с наибольшей лояльностью
- Определены ключевые факторы повторных покупок
- Разработаны рекомендации по удержанию клиентов

## Технологии
- Python
- Pandas
- SQLAlchemy
- Matplotlib/Seaborn
- Phik для корреляционного анализа
